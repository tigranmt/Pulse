Simple API to send data to Pulse.

A simple C# example: 
					
						   Pulse pulse = new Pulse("CLIENT_CODE", "APP_VERSION");            
	            		   
	            		   /*Notify app started*/
	            		   pulse.Start();

	                       //notify some action happen in app and set its value
	            		   pulse.SignalAction("CUSTOM_FACEBOOK_LOGIN_ACTION", "facebook request");
	            		   .....
	            		   pulse.SignalAction("CHECK_ACCOUNT", "checking account");
	            		   ...            

	            		   pulse.Stop();
